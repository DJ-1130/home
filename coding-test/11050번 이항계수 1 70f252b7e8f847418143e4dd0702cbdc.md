# 11050번 : 이항계수 1

## 문제 정보

### 자연수 N과 정수 K가 주어졌을 때 이항 계수 NCK를 구하는 프로그램을 작성하시오

### 입력

첫째 줄에 N과 K가 주어진다.(1≤N≤10, 0≤K≤N)

### 출력

N/K를 출력한다.

### 예제 입력 1

`5 2`

### 예제 출력 1

`10`

## 문제의 의도

이항계수가 무엇인지 알아야 풀 수 있는 문제

### 아이디어

이항계수는 $n!/k!(n-k!)$

### 풀이

(풀이 방법과 코드를 서술한다.)

- Python의 경우

```python
import math
import sys
n, k = map(int, sys.stdin.readline().split())
result = math.factorial(n)//(math.factorial(k)*math.factorial(n-k))

print(result)
```
